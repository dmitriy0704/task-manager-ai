# Проект Spring Boot LLM Ai

Для теста используется сценарий таск трекера.

## Создание задачи

На эндпоинт "/ai/tasks" отправляется DTO TaskRequest(taskTitle,
taskDescription).

### Тестовый запрос/ответ:

1.
 
```text
- "Отправь POST запрос на http://localhost:8090/ai/tasks с JSON телом, где
taskTitle это "Подготовиться к собеседованию на Java-разработчика", а
taskDescription это "Собеседование будет через 3 дня, нужно повторить Spring
Boot и базы данных". Выведи ответ в консоль."

- "Создай задачу через мой API. Название: "Купить молоко", описание: 
"Взять 2 пачки обезжиренного после работы" ".
```

### Ответ:

```json
 {
  "id": 4,
  "taskTitle": "Подготовиться к собеседованию на Java-разработчика",
  "taskDescription": "Собеседование будет через 3 дня, нужно повторить Spring Boot и базы данных"
} 
```
```text
Задача создана — id: 5, taskTitle: "Купить молоко", taskDescription: "Взять 2 пачки обезжиренного после                                                    
     работы".  
```

### Ошибка 404:

```text
Сервер вернул 404 Not Found — endpoint /ai/tasks не существует
на http://localhost:8090. Сервер работает, но такой маршрут не реализован.   
```
